Qualifcations

HTML

A standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.

CSS

A programming design language that includes all relevant information relating to the display of a webpage. CSS defines the style and formatting of a website or page, including the layout, colors, fonts, padding and more.

Sass

Sass is an extension of CSS. It was designed to help developers write instructions on how to present text on a screen rather than to work with variables or perform complex decision-making tasks.

JavaScript

JavaScript is a scripting or programming language that allows you to implement complex features on web pages. JavaScript can display interactive maps, animated 2D/3D graphics, scrolling video jukeboxes, etc.

Java

Java is a high-level, class-based, object-oriented programming language. It is a general-purpose programming language intended to let programmers write once, run anywhere - can run on all platforms that support Java without the need to recompile.

Photoshop

Photoshop is an image creation, graphic design and photo editing software developed by Adobe. The software provides many image editing features for pixel-based images, raster graphics and vector graphics.

VBA

VBA is used to write programs for the Windows operating system and runs as an internal programming language in Microsoft Office (MS Office, Office) applications such as Access, Excel, PowerPoint, Publisher, Word, and Visio.

Lightroom

This software is a successor of Photoshop and much more advanced and updated than it. It is basically an image enhancer and manipulation software and a part of the Adobe software suite.

Adobe Illustrator

Adobe Illustrator is a vector graphics editor and design program developed and marketed by Adobe. Adobe Illustrator was reviewed as the best vector graphics editing program in 2021 by PC Magazine.

Hammer SDK

Hammer Valve Hammer Editor (more informally known as Hammer, and previously called Worldcraft) is the official mapping tool for the GoldSrc, Source and Source 2 engines (which most newer Valve games run under). It is also included in every game made with Source Engine that is not a mod

Vue

Vue.js is an open-source model–view–viewmodel front end JavaScript framework for building user interfaces and single-page applications.
